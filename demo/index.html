Animate css

Doc for css animations

This file contains a collection of keyframe animations and associated classes to apply various animations to icons with the prefix tg-effect-.

The animations include:

  • beat
  • beat-fade
  • bounce
  • fade
  • flip
  • shake
  • spin
  • spin-reverse
  • spin-pulse
  • swing
  • jello
  • rubberBand

Usage in Elements

  • Add the class tg-effect-<animation> to your icon element to apply an animation.
  • Add the class tg-effect-<animation>-hover to apply the animation on hover.

Usage in Parent Animated Elements

  • Add the class tg-parent-hover to your element parent to apply an animation on icon when to hover you.

Modifiers

  • Use .tg-animate-delay-Xs to set animation delay, where X is the delay in seconds [1 to 10].
  • Use .tg-animate-duration-Xs to set animation duration, where X is the duration in seconds [1 to 10].
  • Use .tg-animate-count-X to set the number of animation iterations, where X is the count [0 to 10].

Special Modifiers

  • .tg-animate-delay-faster, .tg-animate-duration-faster: Set time to 0.5s.
  • .tg-animate-delay-fast, .tg-animate-duration-fast: Set time to 0.8s.
  • .tg-animate-count-infinite: Set infinite iterations.
